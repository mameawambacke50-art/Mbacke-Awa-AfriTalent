# AfriTalent
Projet fil rouge — Plateforme de mise en relation entre freelances africains et
clients.

AftiTalent

Presentation
  AfriTalent est une plateforme web qui met en relation les freelances africains et les entreprises a la recherche de competences dans differents domaines du numerique.
  Le projet a ete realise dans le cadre du module de developpement web.
  
  Fonctionalites 
  -Consultation des profils de freelances
  -Presentation des diffenrentes categories de services
  -Consultation des offres tarifaires
  -Presentation de l'entreprise et de son equipe
  -Formulaire de contact 
  -Interface responsive adaptee aux ordinateurs,tablettes et smartphones
